Mari kita belajar mengenai Style Background CSS

Style Background CSS adalah cara Anda dapat mengubah tampilan latar belakang elemen HTML. Anda dapat mengganti warna latar belakang, mengatur gambar latar belakang, mengatur transparansi, dan banyak lagi.

Properti Background CSS

Berikut ini beberapa properti CSS yang sering digunakan untuk mengatur background:

  1. background-color: Mengubah warna latar belakang elemen. Anda dapat menggunakan nilai berupa nama warna (seperti "red" atau "blue") atau kode warna (seperti "#FF0000" untuk merah).
  2. background-image: Menambahkan gambar sebagai latar belakang elemen. Anda perlu menyebutkan path (alamat file) ke gambar yang ingin Anda gunakan.
  3. background-repeat: Mengatur apakah gambar latar belakang akan diulang (repeat) atau tidak. Nilai yang umum digunakan adalah "repeat" (default), "no-repeat," "repeat-x," dan "repeat-y."
